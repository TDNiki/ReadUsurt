# Use Guide

## BBrequest

1. Создайте экземпляр класса, указав в качестве параметра корневую папку расписания. ВАЖНО, чтоб эта папка была общедоступной. Пример: "https://bb.usurt.ru/bbcswebdav/xid-1180291_1"
2. Вызовите функцию get_urls


## ReadSchedule

1. Создайте экземпляр класса, указав в качестве параметра ссылку на эксель файл.
2. Вызовите функцию get_all. У этой функции есть необязательный параметр - date_start_scan: дата начала сканирования расписания.
